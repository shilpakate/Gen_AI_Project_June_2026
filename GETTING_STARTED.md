# 🎓 Getting Started - Step by Step Guide

This guide will walk you through the system from **zero to running**.

## 📋 Prerequisites

You need:
- Python 3.9+
- pip (Python package manager)
- Anthropic API key (get one from https://console.anthropic.com/)
- A terminal/command line

## 🎯 Step-by-Step Setup

### **Step 1: Navigate to the Project**

```bash
cd loan_approval_system
```

### **Step 2: Create a Virtual Environment (Optional but Recommended)**

This keeps your project dependencies isolated from system Python.

```bash
# Create virtual environment
python3 -m venv venv

# Activate it
# On macOS/Linux:
source venv/bin/activate

# On Windows:
# venv\Scripts\activate

# You should see (venv) in your terminal
```

### **Step 3: Install Dependencies**

```bash
pip install -r requirements.txt
```

This installs:
- `fastapi` - Web framework for the backend
- `uvicorn` - Server to run FastAPI
- `streamlit` - Framework for the UI
- `anthropic` - Claude API client
- `fastmcp` - MCP protocol implementation
- `langchain`, `langgraph` - Orchestration tools
- `pydantic` - Data validation
- `python-dotenv` - Environment variables
- `requests` - HTTP client

### **Step 4: Set Your API Key**

Open the `.env` file and add your Anthropic API key:

```bash
# Edit the .env file
nano .env
```

Or open it in your code editor and change:
```
ANTHROPIC_API_KEY=your_actual_api_key_here
```

Get your API key from: https://console.anthropic.com/

### **Step 5: Start the Backend Server**

Open a **NEW terminal** (keep this one running):

```bash
# Make sure you're in the project directory
cd loan_approval_system

# Activate virtual environment if using one
source venv/bin/activate

# Start the FastAPI server
python main.py
```

You should see:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
INFO:     Application startup complete
```

✅ **Backend is running!** Don't close this terminal.

### **Step 6: Start the Streamlit UI**

Open **ANOTHER terminal** (third one):

```bash
# Navigate to project
cd loan_approval_system

# Activate virtual environment
source venv/bin/activate

# Start Streamlit
streamlit run app.py
```

You'll see:
```
  You can now view your Streamlit app in your browser.
  Local URL: http://localhost:8501
```

✅ **The UI should open automatically in your browser!**

## 🏗️ Understanding the Architecture

### **The Flow**

```
User fills form in Streamlit
        ↓
    Sends to FastAPI
        ↓
FastAPI calls multiple agents:
  ├→ Applicant Profile Agent (gets user data)
  ├→ Risk Analysis Agent (analyzes finances)
  ├→ Decision Agent (makes decision + Claude explanation)
  └→ Compliance Agent (logs everything)
        ↓
    Results returned to Streamlit
        ↓
    User sees decision
```

### **Key Components**

1. **Streamlit UI** (`app.py`)
   - What the user sees
   - Three pages: Apply, Check Status, Dashboard

2. **FastAPI Backend** (`main.py`)
   - Receives requests
   - Orchestrates agents
   - Returns decisions

3. **Agents** (`agents/` and `mcp_servers/`)
   - Specialized tools for different tasks
   - Each focuses on one responsibility
   - Use MCP protocol

4. **Claude AI** (`utils/claude_client.py`)
   - Generates human-friendly explanations
   - Makes decisions more understandable

## 💻 Code Walkthrough

### **How a Decision is Made** (simplified)

```python
# 1. Get applicant data
profile = fetch_applicant_profile("APP001")
income = profile["income"]  # $75,000
credit_score = profile["credit_score"]  # 750

# 2. Calculate metrics
income_stability = calculate_income_stability_score(income, employment, years)
# Result: 0.8 (good)

# 3. Analyze risk
dti = calculate_debt_to_income_ratio(income, debts, loan_amount, tenure)
# Result: 0.35 (healthy)

credit_risk = get_credit_risk_level(credit_score)
# Result: "Low" (good)

# 4. Make decision (scoring system)
score = 0
if income_stability > 0.7: score += 1  # ✓ +1
if credit_risk == "Low": score += 1    # ✓ +1
if dti < 0.4: score += 1               # ✓ +1
if employment_risk == "Low": score += 1 # ✓ +1

# score = 4

if score >= 4:
    decision = "Approved"  # ✓
```

## 🐛 Common Issues & Solutions

### **Issue: "Cannot connect to the backend server"**

**Problem**: Streamlit can't find the FastAPI server

**Solutions**:
1. Make sure `python main.py` is running in the other terminal
2. Check that it says `Uvicorn running on http://0.0.0.0:8000`
3. Try accessing `http://localhost:8000/health` in your browser
4. If that works, restart Streamlit: `streamlit run app.py`

### **Issue: "ModuleNotFoundError: No module named 'anthropic'"**

**Problem**: Dependencies not installed

**Solution**:
```bash
pip install -r requirements.txt
```

### **Issue: "ANTHROPIC_API_KEY not found"**

**Problem**: Your API key isn't set

**Solutions**:
1. Check `.env` file exists and has your key
2. Verify the key format is correct
3. If you copied wrong, get a new one from https://console.anthropic.com/
4. Restart both servers after changing `.env`

### **Issue: Port 8000 already in use**

**Problem**: Something else is using port 8000

**Solutions**:
```bash
# Find what's using port 8000
lsof -i :8000

# Kill the process (if it's old FastAPI)
kill -9 <PID>

# Or change the port in main.py:
# uvicorn.run(app, port=8001)
```

### **Issue: "Applicant APP001 not found"**

**Problem**: Using an invalid applicant ID

**Solution**: Use valid IDs: `APP001`, `APP002`, or `APP003`
